Packet Analyzer - README.md
Overview

This project is a simple Packet Analyzer built using Python.
It captures and analyzes network packets to display useful information such as:

Source IP Address
Destination IP Address
Protocol Type
Packet Data

This project helps beginners understand basic network packet sniffing and cybersecurity concepts.

Features
Capture network packets
Display packet details
Identify protocols
Beginner-friendly Python project
Technologies Used
Python 3
Socket Programming
Struct Module
Requirements

Install Python before running the project.

You can download Python from:

Python Official Website

Installation
Clone the repository:
git clone https://github.com/your-username/packet-analyzer.git
Open the project folder:
cd packet-analyzer
Run the program:
python packet_analyzer.py

Destination IP: 142.250.183.78
